Simulation-first search-and-rescue drone intelligence pipeline.

Turns drone-style video and telemetry into geotagged alerts, confirmed tracks, confidence scores, and map-ready GeoJSON.

View GitHub Read Validation View Safety Notes

Problem

Search-and-rescue review quickly becomes noisy.

Search-and-rescue drone footage can produce repeated detections, uncertain location estimates, and large volumes of video to review.

SAR-INTEL TOOLKIT reduces that into structured, reviewable outputs.

Public validation

One documented aerial-person baseline.

The public validation harness evaluates the current detector against VisDrone DET validation imagery using the pedestrian and people categories.

The baseline model was not trained specifically on VisDrone aerial-person imagery, so low recall is expected. This is a public aerial-drone detection sanity check, not an operational SAR benchmark.

Map-ready GeoJSON

Confirmed tracks can be visualized in standard web maps and GIS tools.

The demo map loads a sanitized, public-safe GeoJSON asset derived from a non-empty local run. It preserves representative scoring and track properties while removing runtime-only fields and original location context.

Safety limitations

Public claims stay bounded by actual evidence.

  • Prototype only.
  • Human review required.
  • Not operational SAR software.
  • Geotagging is approximate and configuration-dependent.
  • Pose-aware flat-ground mode is not terrain-aware calibrated photogrammetry.
