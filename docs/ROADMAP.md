# Roadmap

This roadmap is intentionally conservative. It identifies the next areas of engineering work without implying operational readiness or promising a large feature expansion schedule.

## Near-term

- Output schema validation
- Run manifest
- Better camera calibration support

## Longer-term

- Pitch/roll-aware geotagging
- Annotated frame export
- Formal evaluation dataset
- Real drone telemetry adapters

The purpose of this roadmap is to show direction and prioritization, not to commit to delivery dates.
